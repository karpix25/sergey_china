# TikTok Content Manager

Система для автоматизации создания и публикации контента в TikTok с использованием AI.

## Быстрый деплой (Docker)

1.  **Настройка переменных окружения**:
    Убедитесь, что файл `.env` в корне проекта содержит все необходимые ключи:
    - `DATABASE_URL` (внешняя база Postgres)
    - `INTERNAL_API_KEY` (сгенерирован автоматически: `hNRAKvDFPcpeJICTfpgPqcHFodRW5Kbd4vjTHRPlMTg`)
    - `SCRAPE_CREATORS_API_KEY`
    - `ELEVENLABS_API_KEY`
    - `GOOGLE_CREDENTIALS_JSON`

2.  **Запуск**:
    ```bash
    docker-compose up -d --build
    ```

3.  **Доступ**:
    - Панель управления (Frontend): `http://ваш-ip:5173`
    - API (Backend): `http://ваш-ip:8000`

## Безопасность
- API защищено заголовком `X-Internal-API-Key`.
- Фронтенд настроен на автоматическую передачу ключа.
- В `docker-compose.yml` удалена локальная база данных, используется ваше внешнее подключение.

## Особенности
- **Массовое обновление описаний**: Использует AI для адаптации каждого видео под ваш новый шаблон, сохраняя уникальность контента.
- **Оптимизация**: Фронтенд собирается через многоэтапную сборку (Nginx), что обеспечивает высокую скорость работы в продакшене.
